usage: run.py [-h] filename output_sound_name output_image_name jpeg_quality

Convert .wav -> jpg compress -> .wav

positional arguments:
filename           your sound file in .wav format e.g. "test.wav"
output_sound_name  name of your output soundfile e.g. "output.wav"
output_image_name  name of your output jpg image e.g. "output.jpg"
jpeg_quality       quality of jpg compression from 0 - 100

-h, --help         show this help message and exit
